Tropical Cyclone Intensity prediction

over North Indian Ocean using Machine Learning

About TCIP

Tropical cyclone intensity prediction using machine learning involves using statistical and computational techniques to predict the strength and intensity of tropical cyclones, which are powerful and destructive weather events that can cause significant damage to coastal areas and communities.

Machine learning model is trained on historical data of North Indian Ocean tropical cyclones, including their location, Maximum sustained wind speed, estimated central pressure, Pressure drop, sea surface temperature. This model learns to identify patterns in the data and use them to make predictions about future tropical cyclones. The prediction process involves several steps, including data collection, preprocessing, feature engineering, model selection, hyperparameter tuning, and model evaluation.

Tropical cyclone intensity prediction using machine learning has important applications for disaster preparedness and response. Accurate predictions can help to inform evacuation plans, emergency response efforts, and other measures that can mitigate the impact of these powerful storms on communities and infrastructure.
